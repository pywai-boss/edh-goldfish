# UI Rules

## Visual Direction
- Use a dark theme.
- Use neutral panels with subtle borders.
- Use one accent color only: gold/amber.

## Layout
- After parsing, do not keep a sidebar.
- Use full-width analysis as the primary layout.
- Deck controls should be a compact toolbar.

## Components
- Cards should use thin borders.
- Cards should use minimal shadow.
- Cards should use tight padding.
- Stats should use a large number with a small label.

## Typography
- Use a maximum of 3 font sizes.

## Color Usage
- Use mana colors only as small indicators.

## Design Priority
- Prefer density over decoration.
